« What is your opinion ? » est un plugin de commentaires. La différence est qu’il valorise les auteurs et les invite à compléter les articles et à participer à la modération.

== Description ==

« What is your opinion ? » est avant tout, un projet de plugin de commentaires comme il en existe déjà. La différence de notre application est qu’elle valorise les auteurs et les invite à compléter les articles de presse ou de blogs, tout en participant activement à la modération.

== Installation ==

1. Décompressez l'archive dans le dossier `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Pour un fonctionnement optimal, activez le plugin `Disable Comments`
